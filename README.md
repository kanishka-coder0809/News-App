News Website

Overview :

This is a dynamic news website built using HTML, CSS, and JavaScript, integrating a news API to display the latest daily news from various categories. Users can browse trending articles, search for specific news topics, and stay updated with real-time news updates.

Features

✅ Fetches daily news dynamically using a news API

✅ Search functionality to find specific news topics

✅ Category-based filtering (e.g., Technology, Sports, Business, etc.)

✅ Interactive and user-friendly UI

Technologies Used

HTML5

CSS3

JavaScript

News API
